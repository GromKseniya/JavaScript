# Program 1
## Python
```python 
name = input("Введите ваше имя: ")  
print("Привет,", name, "!")
# Вариант 2
print("Привет, ", name, "!", sep="")
# Вариант 3
print("Привет, {}!".format(name))
# Вариант 4
print("Привет, %s!" % name)
# Вариант 5
print(f"Привет, {name}!")
```
## JavaScript
```javascript
let name = prompt("Введите ваше имя: ");
alert("Привет, " + name + "!")
// Вариант 2
alert("Привет, ${name}!");
```
